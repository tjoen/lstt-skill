## Trivia skill with local stt
This skill starts a game of general trivia.

## Description 
At the moment, the skill will ask you 5 questions. You can answer by choosing 1,2,3 or 4.

## Current state
Working features:
- play trivia
- let's play trivia
- game of trivia
